## Run locally

```bash
git clone git@github.com:Soumik205/todo-fullstack.git
cd backend
node Server.js
cd ../frontend
pnpm install
pnpm dev
```

This will start the development server on http://localhost:5173 

*You need to have Node installed and pnpm enabled.*
